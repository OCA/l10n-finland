=============================
Finnish Reconciliation
=============================

Common Finnish account reconciliation.

* Adds a name field to the bank statement form view.
* Adds payment reference to the reconciliation widget
* Makes reconciliation propositions based on payment reference


Installation
============

Install the module form Settings->Local Modules
